# 월간 트랜드 보고서

서브컬처·가챠 게임 6종의 월간 캐릭터 트렌드 분석 보고서.

## 대상 게임

- 원신 (Genshin Impact)
- 붕괴: 스타레일 (Honkai: Star Rail)
- 젠레스 존 제로 (Zenless Zone Zero)
- 명조: 워더링 웨이브 (Wuthering Waves)
- 명일방주: 엔드필드 (Arknights: Endfield)
- 승리의 여신: 니케 (NIKKE)

## 구성

각 월별 HTML 보고서는 다음 섹션으로 구성:

1. **이달의 3대 기획 트렌드**
2. **공통 트렌드 분석** (5 cross-cutting insights)
3. **메커니즘·기믹 비교 매트릭스** (6 games × 5 cols)
4. **Do / Don't 체크리스트**
5. **게임별 카탈로그** — 3개월 픽업 타임라인 + 캐릭터 카드 (한글 메인, 영문 서브, Splash Art)
6. **다음 달 주목 캐릭터 (Watchlist)**

## 방법론 주의사항

- 매출 수치는 Sensor Tower 기반 iOS + Google Play 추정치. 웹스토어·PC·콘솔·중국 로컬 스토어 미집계.
- 티어 평가는 한국·영어 공략 사이트 종합 가중 평균. 공식 지표 아님.
- 유저 평가 3분할(긍정/부정/논란)은 정성 샘플링 기반, 통계적 대표성 없음.
