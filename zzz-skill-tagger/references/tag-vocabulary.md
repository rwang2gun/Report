## 부가효과 태그 전체 목록 (빈도 내림차순)

> 새 태그를 추가할 때는 이 목록에서 가장 유사한 기존 태그를 우선 사용한다.

### 전투 판정 계열
- `무적판정` — 공격 받지 않는 무적 프레임 (가장 많이 쓰임, 411회)
- `자동 회피` — 특정 조건 하에 자동으로 발동하는 회피 (비비안 숙녀의 슬라이딩 스텝 등)
- `패링` — 특정 공격을 막는 패링 판정
- `가드판정` — 가드로 처리되는 판정
- `경직저항증가` — 피격 시 경직 저항
- `반격` — 특정 조건에서 반격 발동

### 피해 증가 계열
- `피해증가` — 일반적인 피해 증가
- `파티원피해증가` — 파티 전체 피해 증가
- `궁극기피해증가` — 궁극기 피해 증가
- `강화특수스킬피해증가` — 강화 특수 스킬 피해 증가
- `속성이상피해증가` — 속성 이상 피해 증가
- `관입 피해증가` — 관입(방어무시) 피해 증가
- `이상피해증가` — 이상 상태 피해 증가
- `받는피해감소` — 받는 피해 감소
- `피해감소` — 피해 감소
- `데미지버프` — 범용 데미지 버프
- `피해량버프` — 피해량 버프

### 그로기/스턴 계열
- `높은그로기배율` — 그로기 수치 누적 배율 높음
- `그로기수치증가` — 그로기 수치 증가 효과
- `그로기배율증가` — 그로기 배율 증가
- `그로기약체화배율증가` — 그로기 상태 약체화 배율 증가
- `그로기` — 그로기 상태 유발

### 스탯 버프 계열
- `공격력증가` — 공격력 버프 (자신)
- `파티공격력증가` — 파티 공격력 버프
- `치명타확률증가` — 치명타 확률 증가
- `치명타피해증가` — 치명타 피해 증가
- `관통력 증가` — 관통력(방어 무시율) 증가
- `관통률증가` — 관통률 증가
- `파티관통률증가` — 파티원 관입력 증가 (반인호 활혈 효과 등)
- `이상축적효율증가` — 속성 이상 축적 효율 증가 (자신)
- `파티원이상축적효율증가` — 파티원 전체 속성 이상 축적 효율 증가
- `이상마스터리증가` — 이상 마스터리 증가
- `충격력증가` — 충격력 증가
- `차지속도증가` — 차지 속도 증가
- `이상 축적 효율 상승` — 이상 축적 효율 상승 (`이상축적효율증가`로 통일 권장)
- `이상 축적 효율 증가` — 이상 축적 효율 증가 (`이상축적효율증가`로 통일 권장)

### 자원 회복 계열
- `HP회복` — HP 회복 (자신 기준; `체력회복`과 통일)
- `파티원 회복` — 파티원 HP 회복 (파티 대상일 때)
- `에너지회복` — 에너지 자원 회복
- `기운회복` — 기운 자원 회복
- `데시벨회복증가` — 데시벨(궁극기 게이지) 회복량 증가
- `꿈세계 게이지 회복` — 꿈세계 게이지 회복
- `HP소모` — HP를 소모하는 스킬
- `지원포인트회복` — 지원 포인트 회복 (세스 궁극기 3pt 회복 등)

## 적 디버프 태그 전체 목록

적 디버프 태그는 에이전트가 **타깃에게 직접 부착하거나 발생시키는 상태·효과**다.
파티원이 발동하는 효과와 연동되는 반응 메커니즘은 적 디버프가 아닌 시너지로 분류한다.

| 태그 | 설명 |
|------|------|
| `빙염` | 미야비 전용 — 적에게 부착, 서릿발 발동 시 서리열파괴로 전환 |
| `갉힘` | 제인 전용 — 제인 공격 명중 시 부착, 강타 발동 시 치명타 효과 연동 |
| `트랩` | 펄크라 전용 — 추가능력 발동 시 적에게 부착, 여진 공격 피해 30% 증가 |
| `달콤한 공포` | 유즈하 전용 — 강화특수스킬/궁극기 강공격 명중 시 적에게 부착, 매 초 스프링클 버스트 공격 발동 |
| `열상` | 버니스 전용 — 특제 연료 상태에서 스킬 명중 시 부여, 파티원 공격 명중 시 잿불 발동 |
| `궤멸` | 라이터 전용 — 사기 분출 상태 종결의 일격 명중 시 부여, 그로기 지속시간 3초 증가 |
